# Колесо Фортуны  Простое и удобное веб-приложение для проведения розыгрышей и принятия случайных решений.  ## Описание Этот проект представляет собой интерактивный барабан, который вращается по нажатию кнопки. Он идеально подходит для тех случаев, когда нужно быстро и честно выбрать победителя или определиться с вариантом действия.
